**Paso 1:**  
Cuando escribimos:

```sql
CREATE TABLE libros (
    id INTEGER PRIMARY KEY,
    name TEXT,
    rating INTEGER
);
```

Estamos **creando una tabla llamada “libros”**.  
Dentro de esa tabla estamos diciendo que tendrá **tres columnas**:

- **id** → número que identifica cada libro (no puede repetirse).
    
- **name** → el nombre del libro.
    
- **rating** → una valoración del libro en número.
    

Cuando escribimos:

```sql
INSERT INTO libros VALUES (1, "Principios de SQL", 1);
```

Estamos **metiendo un libro dentro de la tabla**, indicando su id, su nombre y su valoración.
